# Project: deploys to Netlify via GitHub

When the user wants to **publish, deploy, ship, "go live", get a shareable link, or update their live site**, use the **netlify-deploy** skill in `.claude/skills/`.

NEVER ask the student to run terminal commands or use the Netlify dashboard — run everything yourself with your own tools.

Prefer the **Netlify connector** for the deploy. Only fall back to guiding the student through a browser step if the connector isn't available.

The student speaks only in plain language. You do all the Git and Netlify work.
